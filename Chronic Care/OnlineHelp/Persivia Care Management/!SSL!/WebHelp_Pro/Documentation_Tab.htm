Documentation Tab

The Documentation Tab allows you to add existing documents that will help in coordinating care for the displayed patient. Two of the most frequently uploaded document types are Patient Consent and Patient Education. Uploading a patient consent document is a useful way to move a patient profile toward an Active status. Patient Education documents are also useful.  For example, if a care coordination session is planned for the next month for a patient with type 2 diabetes, a Care Manager or Care Physician may find it practical to place the following Patient Education documents into the space represented by the Documents Tab:

 

  • Diabetes
  • Diabetes Type 1
  • Diabetes Type 2

 

The Documentation Tab makes this process convenient by allowing you to upload these documents from available files.

To add a document specific to the patient you are working with, take the following steps.

 

  • Ensure that you have the correct patient displayed in the Manage Patient Workflow Window.
  • Click the Documentation Tab. The following workspace appears.

 

 

  • Click Add Document to display an upload option

 

  • To change the document type, click the arrow in the Type field.
  • Browse to locate the desired document file.
  • Ensure that it is of one of the supported types (.pdf, .doc, .docx, .xls, or .xlsx).
  • Click Save.  A message appears at the upper right telling you that you saved the document successfully.

 

Note that when you save the upload, the CM system creates a record of the save. This record then updates the Change Log with the following display:

 

 

The Documentation Tab cannot be used to draft notes; it is used only to upload documents that are already on file. You would use the Notes tab to make notes or record current observations.
