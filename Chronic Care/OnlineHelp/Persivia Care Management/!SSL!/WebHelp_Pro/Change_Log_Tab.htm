Change Log Tab

The Change Log records diverse events, capturing them within a list of types. The Types appear in the Event Column.  The Change Log is a tab that you can view but not directly alter.  The Change Log records every action that is performed on behalf of the patient who is currently displayed in the Workflow Window.  For example, a patient with a substantial history in the system might have a change-log display that looks like this:

 

 

Because the Change Log captures many events without requiring action from the user, the Change Log can function as a comprehensive prompt. If you are investigating an alert, for example, it can be helpful to view the Change Log to locate a dense record of events for the time period.
