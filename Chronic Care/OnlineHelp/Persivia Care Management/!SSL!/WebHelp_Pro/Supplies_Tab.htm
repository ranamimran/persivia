Supplies Tab

A full care plan should specify any clinical supplies that are needed Requirements can differ according to special patients’ needs, but some information about supplies can be derived from the diagnosis.  CM provides the Supplies Tab for detailing this information.  

 

To display the Supplies Tab, select the patient from the Manage Patient Workflow and display his or her information in the Workflow Window. Then click the Supplies tab in the resulting workflow window. The Supplies Tab opens, as shown:

 

 

The Supplies Tab contains the following fields and displays them as column heads to allow for additions in each category:

 

  • Supply Name: The common descriptive name for the item being ordered or recommended
  • Type: The category of supply used. One of the available types is “OTC Consumables" (over-the-counter consumables). These items are disposable and are often aids to personal care, such as bandages.  The other category is DME, Durable Medical Equipment. These items can be used over time and include crutches and other walking aids.
  • Model: The model name is used to help specify the functional requirements that the item must meet, and to discourage substitution.
  • Manufacturer: The name of the manufacturer of the supply item
  • Serial Number: The serial number of the item or the device. The serial number will only be used for items of durable medical equipment.
  • Count: the number of units recommended for the patient
  • Created On: The date the request or order for the item was given. If you are ordering a new item, this date might be today’s date.
  • Start Date: The date the supply started being used by the patient
  • End Date: The date the supply was no longer being used by the patient
  • Status: The status of the item. The options will be "Assigned", "Installed", and "Uninstalled".
  • Notes: open text box where the User can manually add any additional information
