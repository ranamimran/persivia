Interventions Tab

The Interventions Tab lists actions or care events.  These actions or care events that have been planned or completed for the patient who is displayed in the current workflow window. The Interventions Tab appears in the following format:

 

 

The window lists details about each intervention as column heads. They are as follows:

 

  • Intervention: A description of the action requested, usually phrased as an instruction to do something.  No grammatical time indicators are needed. Whether the intervention is in the past, present, or future can only be determined by the entry in the Status column
  • Created on:  The date that the intervention was created in the system. You cannot change this date if you are viewing an existing investigation.
  • Target: The target date for the intervention to be completed
  • Priority: Low, Medium, or High. If you are adding a new intervention, CCM provides a drop-down menu with these selections
  • Status: Active, Pending, or Completed you can change these values if you are adding or updating an intervention. However, you cannot change the Status of a completed intervention. If action of this type is again needed, enter a new intervention

 

To add an intervention, click the +Add Interventions button.  A popup window appears listing the fields that will inform the display columns when you have saved your changes.
