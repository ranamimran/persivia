Tabs and Field Information

The tabs in this window provide a central point for each patient, yet they link to a wide range of care-coordination tasks.  These tabs also put the information required for care coordination in a single place.

Patient Tab

The Patient Tab is open by default when you open the MPW Window; it lets you scan (and add to) contact information about the patient and any associated guardian or care giver.

 

 

The Patient Tab displays the following information:

  • Patient: Identifies the patient by CMID.
  • MRN: The patient’s medical record number
  • DOB: The patient’s date of birth
  • Age: The patient’s current age in years
  • Gender: Male, Female or Unspecified
  • Care Manager: The name of the currently assigned care manager, if any
  • Physician: The care physician who reviews care plans for this patient.

 

In the Support Persons area, the screen displays these details:

  • Last (then First) Name: Name of Support Person
  • Relationship: Indicates familial relationship, if any
  • Start Date: Date when this person was first listed as a supporter
  • End Date: If any, the date when the listed person’s role as a support person ended. This date would also end the support person’s right to access the patient’s records. (The Add button on this screen allows you to add a new support person at this time if indicated).
