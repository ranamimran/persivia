Graphics and Navigational Tools Common across CM

Each tab has a button that allows you to add a new instance of the element documented by the tab. For example, the Care Plan tab has a button on the upper left, +New Care Plan.  It allows the Care Manager or Care Physician to add a new care plan to the patient profile. Similarly, the Assessment tab has a button on the upper left called +New Assessments. The Goals tab has a button near the upper left that says, +Add Goals, as shown in Figure 9. Features Common across Workflow Tabs. You can use the +Add button or the +New button to add new information whether you are adding a patient (or Care Team member) for the first time or using the Workflow Window to manage active patients.

 

Another feature that is common across the set of care plan elements (such as treatment goals) details is a pair of columns on the right –hand edge of the screen. They contain buttons that allow you to edit or delete the selected element

 

 

The tabs characteristically contain a column each for Notes, Status, and information about start and end dates.  In addition, each tab allows you to sort the display by column-head in ascending or descending order. Finally, many column heads, both in the Manage Patient Workflow and the Workflow Window, display a column-selection menu that lets you select which columns appear in the tab. This option makes it easier to organize the information you need to take an action such as approving a care plan.
