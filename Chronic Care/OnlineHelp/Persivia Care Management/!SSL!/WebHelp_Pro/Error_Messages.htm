Care Management Error Messages

There are many Error Messages that may be produced by a Care Management unit. This topic is intended to list the messages and provide reasons and suggested actions.

Error Reason Action
1001 Low battery Replace battery
1010 Overheating Cool unit
2035 Signal Weak Possible obstruction with antenna - Move obstruction
3300 No Signal Dead battery or damaged antenna - Replace battery or antenna

 

This is an example of an Error Messages page. Note the bookmark flags. Each possible error message would be listed in the Index. This would allow the help user to easily locate information related to the error.
